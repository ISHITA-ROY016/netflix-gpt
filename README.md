 # Features
  
  - Login/Signup page
    - Sign in/ Sign up form
    - redirect to browser page
  - BrowsePage (after authentication)
    - Header
    - Main movie
      - Trailer in bg
      - Title & description
    - Movie suggestion
      - MoviesLists * N 
  - NetflixGPT 
    - Search bar
    - Movie suggestions
   

  ## APIs used 
   - Streaming Availability ([official doc](https://docs.movieofthenight.com/))
   - IMDB ([rapidapi.com](https://rapidapi.com/hub))
